# Первый файл по контролю версий.
Для добавления заголовка поставьте # в начале строки.
## Выделение текста.
Для выделения разделов текста поставьте ## в начале строки.

*Курсив.*

*Для выделения текста курсивом обрамите его звездочками.*

**Полужирный.**

Для написания текста полужирным шрифтом обрамите его ** с двух сторон.

***Курсивный полужирный шрифт.***

Чтобы создать курсивный полужирный шрифт обрамите его тройными звездочками.

~~Зачеркнутый текст.~~

Для создания зачеркнутого текста обрамите его двойными тильдами.

## Списки

Для создания ненумерованного списка поставьте перед ним * и пробел.

* Элемент 1
* Элемент 2
* Элемент 3

Для создания нумерованного списка поставьте перед ним номер с точкой и пробел.

1. Первый элемент нумерованного списка.
2. Второй элемент нумерованного списка.

    * первый элемент
    * второй элемент
    * третий элемент

Чтобы создать вложенный список сделайте отступ в начале строки.
